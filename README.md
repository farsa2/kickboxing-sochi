# Kickboxing Sochi — Website

Commercial website for a kickboxing club in Sochi.

## 🚀 Tech Stack

- JavaScript (ES6+)
- HTML5 / CSS3
- Vite
- REST API

## ✨ Features

- Responsive layout (mobile / tablet / desktop)
- Media content (images, videos)
- API integration
- Optimized asset loading
- Production-ready build

## 🔗 Live Demo

https://kickboxing-sochi-api.vercel.app

## 📂 Project Structure

- /api — API logic
- /assets — styles
- /images — images
- /videos — media content

## 📌 About

This project was developed for a real client as a commercial website.
